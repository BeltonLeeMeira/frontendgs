AlertaFácil - Solução para Enchentes no Brasil

O AlertaFácil é um sistema inteligente de alerta para enchentes, desenvolvido para prevenir tragédias anunciadas no Brasil. Este projeto visa proteger a população em áreas de risco, agentes da Defesa Civil e prefeituras municipais, utilizando tecnologia acessível e escalável.

O Problema

O Brasil sofre anualmente com enchentes devastadoras causadas por chuvas intensas, falta de drenagem e ausência de monitoramento adequado. Muitas vezes, a população só é alertada quando o problema já se agravou, resultando em perdas de casas, bens e vidas.

Como a Solução Funciona

O sistema AlertaFácil é composto por sensores de nível de água em pontos críticos, um app de alerta para celulares com mensagens automáticas baseadas em localização, um dashboard para a Defesa Civil acompanhar os dados em tempo real, e um sistema em Python que simula histórico, gera alertas e analisa riscos.

Diferenciais

  • Envio de alertas automáticos baseados em localização
  • Dashboard visual para monitoramento em tempo real
  • Funcionamento parcial offline para maior confiabilidade
  • Sistema modular com partes independentes
  • Design atrativo, acessível e intuitivo
  • Responsividade total com abordagem Mobile First
  • Uso de CSS Grid Layout para organização
  • Micro interações e animações para dinamismo visual
  • Interação com JavaScript para melhor experiência do usuário

Backlog e Casos de Uso

O projeto possui um backlog de funcionalidades que inclui monitoramento via sensor, envio de dados via MQTT, análise de risco em Python, notificações via app, mapa com rotas de fuga e dashboard com dados ao vivo.

  • UC01 - Monitoramento de Enchente: Sensor coleta dados, envia para dashboard, gera alertas
  • UC02 - Alerta ao Usuário: App recebe alertas push com rotas de evacuação

Protótipos e UX Design

O design do app prioriza acessibilidade, clareza e urgência, com textos grandes, alto contraste e interface simples para facilitar o uso em situações de emergência.

  • Acessibilidade: textos grandes, contraste alto, poucos cliques
  • Clareza: interface com ícones e frases curtas
  • Urgência: destaque em vermelho para nível de risco

O design é simples e direto, pensado para funcionar bem em regiões com pouca conectividade e para pessoas com baixa alfabetização tecnológica.
